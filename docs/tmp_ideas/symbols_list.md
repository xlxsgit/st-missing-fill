# 主要符号表

本表汇集了在整篇本论文中跨越各个章节统一使用的主体数学与物理符号，以供快速查阅对齐。若个别局部算子或非显式系数在正文中带有专用附注，则均以其局部定义为主。

| 符号 | 意义说明 | 单位 / 值域空间 |
|:---|:---|:---|
| $\mathcal{S}$ | 目标时空研究区域内所有气象监测站点的全集 | 无量纲集合 |
| $S$ | $\mathcal{S}$ 中包含的监测空间站点总个数标量 | 个 (标量) |
| $T$ | 时间序列观测的时窗跨度总步数或总时间节点极限 | 步 / 时间刻度参量 |
| $\mathbf{Y}$ | 时空响应变量（如PM2.5或风速）的包含缺失观测的历史实测矩阵 | $\mathbb{R}^{S \times T}$ / 依据气态物理量而定 |
| $\mathbf{M}$ | 矩阵 $\mathbf{Y}$ 对应相同尺寸的观测状态二元掩码矩阵指示器（1:缺失, 0:未缺失） | $\{0,1\}^{S \times T}$ |
| $\widetilde{\mathbf{Y}}$ | 未知但存在于现实客观环境中的全区域满秩无损完形数据集（Ground Truth）| $\mathbb{R}^{S \times T}$ / 依据气态物理量而定 |
| $\widehat{\mathbf{Y}}$ | 由模型输出算法推理填补完整后的预测插值状态张量矩阵 | $\mathbb{R}^{S \times T}$ / 依据气态物理量而定 |
| $\boldsymbol{y}_s$, $y_{s,t}$ | 第 $s$ 个站点的时序记录列向量；及时刻 $t$ 此站点的具体标量观测值 | 列向量 / 实数标量 |
| $m_{s,t}$ | 定义 $y_{s,t}$ 是否成功收集的单一布尔属性（本论文记缺失为 1）| $\{0, 1\}$ |
| $L$ | 时间序列预测或双向自回归插补的最大时间滞后或超前切片步阶（Lag阶数） | 整数步 |
| $l$ | 在 $[-L, L]$ 滑动窗内具体指向的一个相对滞后游标偏移标量 | 整数步 $l \in [-L, L]$ |
| $\tau_{s',s}(t)$ | 刻画因果风场流体从上游站点 $s'$ 物理抵达下游 $s$ 的所需阻力耗时 | 归一化时间步频数 |
| $D_{s',s}$ | 球面空间二维平面下两实体站点 $s$ 与 $s'$ 投影的欧氏间距 | 千米 (km) / 米 (m) |
| $\Theta_{s, t}$ | 站点 $s$ 于时刻 $t$ 观测记录到的水平风向动力角度（正北为参考极零度） | 度 ($^\circ$), $0 \sim 360$ |
| $\Phi_{s',s}$ | 站点 $s'$ 朝向站点 $s$ 的绝对地理方位角设定 | 度 ($^\circ$), $0 \sim 360$ |
| $\Lambda_s$ | 依据物理高程测算的地形固有最大倾斜下沉法向量对应的坡向角 (Aspect) | 度 ($^\circ$), $0 \sim 360$ |
| $Z_s$ | 处于同一参照系海平面水平线上测站 $s$ 的绝对高程海拔 | 米 (m) |
| $X_s, Y_s$ | 测站 $s$ 所在局部的地理地图映射二元经度 / 纬度投影坐标标卡 | 无量纲坐标轴数值 |
| $\Delta \theta_{s',s}(t)$ | 风向偏离角：实体风向 $\Theta$ 和映射方位 $\Phi$ 最短夹角差绝对值 | 度 ($^\circ$), $0 \sim 180$ |
| $\mu_{s',s}(t)$ | 基于站台微距背风坡面 / 迎风坡面倾角 $\Lambda$ 的瞬时山体微尺度截断阻尼增益值 | $[0, 1]$ 标量极限 |
| $\tilde{\alpha}_{s',s}(t)$ | 整合所有距离衰减、高度切变与迎风角的综合物理对齐静态时不变常流图传导权 | $[0, 1]$ 单调阈值 |
| $v^{\text{eff}}_{s',s}(t)$ | 向着目标传播轴向上衰变成行的时空物理有效靶流速分量极限估量 | 依据气流物理体系 ($m/s$) |
| $\Omega(l, \tau_{s',s}(t))$ | 指数型基于高斯平滑基衰减度量的上游风流到达靶点精准时间对齐匹配核函数 | $[0, 1]$ 权重 |
| $\mathbf{X}$ | 研究中除主响应物外囊括温度、气压 $P$ 个并行通道复合的同步气象辅助协变量 | $\mathbb{R}^{S \times T \times P}$ |
| $\alpha_l(t), \beta_l(t)$ | 第三章由于不设固定距离算式所依赖待估算求解用以滑动相乘的时间变异系数 | 实数 $\mathbb{R}$ |
| $A_{s,l}(t), \beta_{s,l}(t)$ | 第四章被时空降维超网络（HyperNetwork）所散离派发的时间自回归保留参数项权 | $[-1, 1]$ 门限内 |
| $B_{s, p}(t), \gamma_{s}(t)$ | 第四章超维分管调控的协变量因果缩放调节器参数项以及 $l=0$ 极近自旋即时权 | $[-1, 1]$ 门限内 |
| $G(\cdot), D(\cdot)$ | 第两章/第三章论证深度生成对抗时代表造假生成器以及边界裁决判别器的映射族 | 深度非线性复合网络系统 |
| $\boldsymbol{b}_q$, $B_{q, p}(z_t)$ | 近似函数参数矩阵重构建用的 B-样条逼近离散有限系数及底层支持分段多项式基底 | 基函数参数域 |
| $\lambda$ | 用于双零和对抗对局动态权衡生成 MSE 残差主损失与辅助隐识别分项占比重衰减率 | 固定小数值 ($<1$) |
| $\Sigma_d, \Sigma_z, \Sigma_{\tau}$ | 第四章模型中随全盘图网格预演解算自我优化的平滑核、气压核及距离落差阻尼标尺 | 无量纲平滑指数 |
