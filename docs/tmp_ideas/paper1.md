# 第X章 基于VCAAN的PM2.5时空缺失值填补研究

本章围绕论文 *Imputation for spatiotemporal PM2.5 data via Varying-Coefficient Autoregressive Adversarial Network* 展开，系统介绍其数据来源、模型设计与实验结果，并总结该方法在复杂缺失机制下的适用性与局限性。

## 1. 数据集介绍

### 1.1 数据来源与研究对象
本文研究对象为北京市 PM2.5 时空监测数据，数据来自北京市生态环境监测中心的 12 个国控监测站点，包括中心城区 8 个站点（Aotizhongxin、Dongsi、Guanyuan、Gucheng、Nongzhanguan、Tiantan、Wanliu、Wanshouxigong），西北部 2 个站点（Changping、Dingling）以及东北部 2 个站点（Huairou、Shunyi）。

### 1.2 数据规模与统计特征
数据时间范围为 2013 年 3 月 1 日至 2013 年 3 月 30 日，采样频率为逐小时，共 8640 条观测。PM2.5 原始浓度范围为 3-558 \(\mu g/m^3\)，原始数据中约有 0.53% 的缺失值。统计描述显示，各站点数据具有一定重尾特征，四分位区间约为 26-152 \(\mu g/m^3\)。

### 1.3 预处理与任务定义
为减小不同站点量纲差异，论文对各站点数据进行 Z-score 标准化。设观测矩阵为 \(Y\in\mathbb{R}^{S\times T}\)，缺失指示矩阵为 \(M\in\{0,1\}^{S\times T}\)（缺失位置取 1），目标是估计完整矩阵 \(\hat{Y}\)，并在缺失位置最小化均方误差（MSE）：

\[
\mathrm{MSE}=\frac{\sum_{s=1}^{S}\sum_{t=1}^{T}\left(\hat y_{s,t}-y_{s,t}^{*}\right)^2 m_{s,t}}{\sum_{s=1}^{S}\sum_{t=1}^{T} m_{s,t}}
\]

其中 \(y_{s,t}^{*}\) 为真实值。

## 2. VCAAN 模型

### 2.1 总体框架
VCAAN（Varying-Coefficient Autoregressive Adversarial Network）由生成端 VCA 与判别端 CDN 构成。输入包括不完整数据和预插补数据（文中默认采用 LOCF 预插补）。模型通过“生成-判别”对抗迭代，使插补结果在保持观测点一致性的同时，更接近真实时空分布。

### 2.2 生成器：变系数自回归模型（VCA）
VCA 的核心思想是：同时利用目标站点的历史/未来信息以及邻近站点的时空信息，并允许回归系数随时间动态变化。站点间权重由 Pearson 相关系数给出（同站点自环权重置 0 以避免信息泄露）。

为降低参数维度并刻画非平稳动态，模型将时间变系数用 B-spline 基函数展开；通过估计 spline 系数向量完成动态回归建模。该设计相比常系数 VAR 更灵活，同时在参数规模上可控。

### 2.3 判别器：卷积判别网络（CDN）
CDN 以连续时间窗作为输入样本，文中窗口长度设为 24（即 1 天）。网络结构为 3 个卷积块 + 全连接层：

- 卷积块：Conv1d（核大小 3，步长 1，padding 1）+ BatchNorm + ReLU + MaxPool
- 通道数逐层增大：32 \(\rightarrow\) 64 \(\rightarrow\) 128
- 全连接层：首层 ReLU + Dropout，输出层 Sigmoid

CDN 输出各站点“该段数据为插补值”的概率，用于反向约束 VCA 生成结果。

### 2.4 对抗优化与动态损失加权
训练分为两阶段：

1. 先独立训练 VCA（基于观测位置重构误差）。
2. 再进行 VCA 与 CDN 的对抗训练。

总损失写为：
\[
L_{\text{total}} = L_{\text{VCA}} - \lambda L_{\text{CDN}}
\]

其中 \(\lambda\) 为动态对抗权重。论文通过约束 \(\lambda L_{\text{CDN}}/L_{\text{VCA}}\) 的量级，按步长自适应调整 \(\lambda\)，以缓解两类损失尺度不一致导致的训练不稳定和过拟合风险。

## 3. 实验分析

### 3.1 实验设置

#### 3.1.1 对比方法
论文将 VCAAN 与 7 类基线模型比较：MICE、KNNI、LOCF、GRUD、SSGAN、SAITS、iTransformer。

#### 3.1.2 缺失率设置
考虑 5 档缺失率：10%、30%、50%、70%、90%。由于 MIS 构造限制，MIS 场景中高缺失率以 55%、60% 代替 70%、90%。

#### 3.1.3 缺失机制/模式
实验覆盖 5 类缺失场景：

- 机制类：MCAR、MAR、MNAR
- 模式类：MIS（连续条带缺失，段长 \(l=12\)）、MIB（块状缺失，块边长 \(d=7\)）

其中 MAR 通过时间强度函数模拟，MNAR 通过“高值更易缺失”规则模拟，较贴近空气污染监测中的实际异常。

### 3.2 主结果分析
论文结果表明：

- 在 MCAR、MAR、MIB 三类场景中，VCAAN 在各缺失率下总体表现最佳。
- 在 MNAR 场景中，VCAAN通常位列前两名；在 MIS 场景中除个别缺失率（50%）外也基本位列前两名。
- 当缺失率较高（>50%）或缺失机制更复杂时，VCAAN 相比基线方法优势更明显。
- 固定缺失率下，所有模型性能大体呈 MCAR \(\rightarrow\) MAR \(\rightarrow\) MNAR 递减趋势，说明机制复杂度是影响插补难度的关键因素。

### 3.3 消融实验分析
论文构建了 VCA、CCA、CCAAN、VCAAN 等变体，并测试不同预插补算法对最终结果的影响。结论包括：

- 在 MCAR、MAR 及部分 MIB 场景下，完整模型 VCAAN 普遍优于不含对抗或不含变系数的简化版本。
- VCA 与 CDN 均对性能提升有正向贡献，且两者组合通常效果最稳。
- 在 MNAR 与高缺失场景下，模型对初始值更敏感，说明“预插补依赖”问题仍未完全消除。

按 MSE 改善率统计，VCAAN 的中位提升约为 38.93%，75% 分位提升约为 53.32%，最大提升约为 81.88%，但最差情况下仍出现负提升（约 -13.71%），反映其在极端场景下仍有改进空间。

## 4. 本章小结

本章介绍了基于 VCAAN 的 PM2.5 时空缺失值填补方法。该方法通过“变系数自回归 + 卷积判别对抗”的混合建模思路，兼顾了统计模型可解释性与深度模型表达能力，在多缺失机制、多缺失率条件下表现出较好的精度与鲁棒性。实验表明，VCAAN 在多数场景下优于传统统计方法与主流深度学习基线，尤其在高缺失率和复杂缺失模式下优势更为明显。

同时，模型在 MNAR 与部分连续缺失场景中仍表现出对预插补初值的敏感性，提示后续研究可从引入气象/设备状态等外生信息、弱化初值依赖与提升大规模站点泛化能力等方向进一步优化。
