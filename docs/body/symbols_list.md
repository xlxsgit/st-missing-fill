# 主要符号表（统一版）

下表给出全文统一符号。若局部小节出现临时符号，以该小节首次定义为准，但不得改变下表中核心符号含义。

## A. 索引、维度与集合

| 符号 | 含义 | 取值/维度 |
|:---|:---|:---|
| $\mathcal{S}$ | 站点集合 | $\{1,\dots,S\}$ |
| $s,s'$ | 站点索引 | $s,s'\in\mathcal{S}$ |
| $t$ | 时间索引 | $t=1,\dots,T$ |
| $l$ | 滞后/超前索引 | $l\in\{-L,\dots,-1,1,\dots,L\}$ |
| $p$ | 协变量索引 | $p=1,\dots,P$ |
| $q$ | 展开项索引 | $q=1,\dots,Q$ |
| $S$ | 站点总数 | 正整数 |
| $T$ | 时间步总数 | 正整数 |
| $L$ | 最大滞后阶数 | 正整数 |
| $P$ | 动态协变量个数 | 正整数 |
| $D_z$ | 静态地形特征维度 | 正整数 |

## B. 数据与掩码

| 符号 | 含义 | 取值/维度 |
|:---|:---|:---|
| $\mathbf{Y}$ | 含缺失观测矩阵 | $\mathbb{R}^{S\times T}$ |
| $\widetilde{\mathbf{Y}}$ | 完整真实矩阵（ground truth） | $\mathbb{R}^{S\times T}$ |
| $\widehat{\mathbf{Y}}$ | 最终插补结果 | $\mathbb{R}^{S\times T}$ |
| $y_{s,t}$ | 站点 $s$ 在时刻 $t$ 的观测值 | 标量 |
| $\hat y_{s,t}$ | 对 $y_{s,t}$ 的估计值 | 标量 |
| $\tilde y_{s,t}$ | 对应真实值 | 标量 |
| $\mathbf{R}=(r_{s,t})$ | 观测指示矩阵（观测=1，缺失=0） | $\{0,1\}^{S\times T}$ |
| $\mathbf{M}=(m_{s,t})$ | 缺失指示矩阵，$\mathbf{M}=\mathbf{1}-\mathbf{R}$ | $\{0,1\}^{S\times T}$ |
| $\mathbf{X}$ | 动态协变量张量 | $\mathbb{R}^{S\times T\times P}$ |
| $\mathbf{Z}$ | 静态地形特征矩阵 | $\mathbb{R}^{S\times D_z}$ |

## C. 第三章（VCAAN）相关符号

| 符号 | 含义 | 取值/维度 |
|:---|:---|:---|
| $w_{s,s'}$ | 站点静态空间权重（相关系数构造） | 实数，且 $w_{s,s}=0$ |
| $\alpha_l(t)$ | VCA 时间项变系数 | 实数 |
| $\beta_l(t)$ | VCA 空间项变系数 | 实数 |
| $x^{q,(i)}_{s,t}$ | 第 $i$ 轮迭代中的展开特征 | 实数 |
| $\boldsymbol{\Psi}(z_t)$ | 样条基函数向量 | 向量 |
| $z_t$ | 归一化时间 | $t/(T+1)$ |
| $\boldsymbol{\eta}_q$ | 第 $q$ 个展开项的系数向量 | 向量 |
| $\varepsilon_{s,t}$ | 随机误差项 | 常设为零均值 |
| $\mathcal{G}$ | 生成器（VCA） | 映射 |
| $\mathcal{D}$ | 判别器（CDN） | 映射 |
| $\lambda_{\text{ad}}$ | 对抗损失权重 | 实数 |
| $\delta_{\lambda}$ | 对抗权重调整步长 | 正实数 |
| $\tau$ | CDN 时间窗长度 | 正整数 |

## D. 第四章（PG-STAT）相关符号

| 符号 | 含义 | 取值/维度 |
|:---|:---|:---|
| $D_{s',s}$ | 站点间平面距离 | 米（m） |
| $\Phi_{s',s}$ | 从 $s'$ 指向 $s$ 的方位角 | $[0,360)$ 度 |
| $\Theta_{s,t}$ | 站点风向角 | $[0,360)$ 度 |
| $\Lambda_s$ | 站点坡向角（aspect） | $[0,360)$ 度 |
| $\Delta\theta_{s',s}(t)$ | 风向与连线方位夹角 | $[0,180]$ 度 |
| $\mu_{s',s}(t)$ | 地形迎风增益项 | $[0,1]$ |
| $\tilde\alpha_{s',s}(t)$ | 物理对齐权重 | 非负实数 |
| $v^{\text{eff}}_{s',s}(t)$ | 有效传播风速 | m/s |
| $\tau_{s',s}(t)$ | 传播时延（以步计） | 非负实数 |
| $\Omega(l,\tau)$ | 时空对齐核函数 | $(0,1]$ |
| $A_{s,l}(t)$ | 双向时间项系数 | 实数 |
| $\beta^{\text{sp}}_{s,l}(t)$ | 空间传播项系数 | 实数 |
| $\gamma_s(t)$ | 同时刻空间项（$l=0$）系数 | 实数 |
| $B_{s,p}(t)$ | 协变量项系数 | 实数 |
| $y^{\text{clean}}_{s,t}$ | 超网络输出的清洗值 | 实数 |
| $\tilde y^*_{s,t}$ | 混合清洗值（观测值/清洗值融合） | 实数 |
| $\lambda_{\text{clean}}$ | 清洗头损失权重 | 正实数 |

## E. 评价指标

| 符号 | 含义 | 说明 |
|:---|:---|:---|
| MSE | 均方误差 | 第三章主指标 |
| RMSE | 均方根误差 | 第四章主指标 |
| PI | 相对改进率 | 与基线模型对比时使用 |
