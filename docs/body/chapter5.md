# 第五章 结论与展望

## 5.1 研究结论

本文围绕时空序列缺失值插补问题，针对 PM2.5 与风速两个典型环境变量，完成了统计模型与深度模型融合框架的研究。主要结论如下：

1. 在 PM2.5 任务中，VCAAN 通过“变系数自回归 + 对抗判别”实现了可解释统计建模与分布约束的结合，在多缺失机制下表现出较强稳定性。
2. 在风速任务中，PG-STAT 将地形、风向和传播时延构造成物理先验，并通过时空对齐 Transformer 动态生成方程系数，在多数复杂缺失场景中取得更优 RMSE。
3. 从方法路线看，本文验证了“白盒方程主干 + 深度参数生成/分布约束”的可行性：既保留统计解释性，又提升高维非线性场景性能。

## 5.2 主要创新点

本文的创新点可概括为三方面：

1. 提出统计与深度融合的两级技术路线：第三章从对抗约束角度增强变系数自回归，第四章进一步引入物理先验与超网络参数化。
2. 在风速插补中构建可学习物理对齐机制（方向、距离、高程、时延），有效提升空间相关缺失场景下的恢复能力。
3. 建立统一符号体系与章节化实验框架，便于模型复现实验与后续扩展。

## 5.3 不足与后续工作

尽管本文工作取得阶段性结果，仍存在以下不足：

1. 第三章模型在 MNAR 与长序列缺失场景下仍存在一定初值敏感性。
2. 第四章实验目前以 RMSE 为主，评价维度仍可扩展到 MAE、分位数误差与不确定性校准。
3. 当前风速实验虽覆盖多模式缺失，但对跨季节、跨区域迁移泛化的验证仍不充分。

后续可从以下方向开展：

1. 构建弱初值依赖的联合优化机制，降低极端缺失场景下误差波动。
2. 融合更多外生信息（设备状态、再分析资料、数值天气预报场）提升 MNAR 场景表现。
3. 引入概率插补与置信区间输出，支持风险感知决策。
4. 进一步优化大规模站点计算效率，形成可部署的业务化插补流程。
