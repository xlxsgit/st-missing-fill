# 第一章 绪论

## 1.1 研究背景及意义

在环境监测与气象观测业务中，时空序列缺失是长期存在且难以避免的问题。空气污染监测网络中常见的设备维护、传感器老化、通信中断，会直接导致 PM2.5 等污染物序列出现间断；复杂地形下的风速观测同样受停电、链路故障和恶劣天气影响，形成连续缺测或区域性缺测。若不进行高质量插补，后续的污染溯源、扩散模拟、风险预警与政策评估都将受到系统性偏差影响。

传统均值填补、前向填补等方法实现简单，但难以保留时空依赖结构；纯深度学习方法虽然表达能力强，却往往解释性不足、对样本分布漂移敏感。统计学视角强调“结构可解释 + 参数可推断”，深度学习视角强调“高维非线性 + 表征学习”，二者融合是时空缺失插补的重要发展方向。

本论文围绕“统计结构约束下的深度插补模型”展开研究，分别针对 PM2.5 与风速两类典型场景提出方法：

1. 面向空气污染数据，构建变系数自回归与对抗学习融合的 VCAAN 模型。
2. 面向复杂地形风速数据，构建物理先验引导的时空对齐 Transformer 模型（PG-STAT）。

上述研究具有两方面意义：

1. 理论意义：推动变系数时空统计模型与深度神经网络的可解释融合。
2. 应用意义：提升高缺失率与复杂缺失模式下环境监测数据可用性，为后续预测和决策提供更可靠数据基础。

## 1.2 国内外研究现状

### 1.2.1 时空数据缺失值插补研究现状

缺失数据研究通常从缺失机理与缺失形态两条线展开。按机理可分为完全随机缺失（MCAR）、随机缺失（MAR）与非随机缺失（MNAR）；按形态可分为点状随机缺失、连续序列缺失、空间块状缺失等。相比独立同分布表格数据，时空数据额外包含时间相关性、空间相关性与跨变量耦合关系，使得插补问题更复杂。

现有研究普遍认同两点：

1. 当缺失率较低且机理简单时，多数方法差异不大。
2. 当缺失率升高或出现连续/块状缺失时，是否显式建模时空结构成为性能分水岭。

### 1.2.2 统计方法插补研究现状

统计插补方法以可解释性和稳定性见长，典型方法包括 LOCF、KNN、MICE 及其扩展形式。对于时空序列，向量自回归（VAR）及其变体能够利用历史信息和空间关联完成缺失重构，但常系数假设难以刻画非平稳过程。为此，变系数自回归、样条展开、状态空间模型等被用于增强模型灵活性。

该方向的主要优势在于参数含义清晰、便于推断和诊断；主要不足在于面对强非线性和高维复杂依赖时表达能力受限。

### 1.2.3 深度学习插补研究现状

深度学习方法通过端到端训练学习复杂模式，近年来在时间序列插补上发展迅速。代表性路线包括：

1. RNN/GRU 类：利用递归结构处理时间依赖。
2. GAN 类：通过生成器-判别器对抗提升插补分布真实性。
3. Transformer 类：通过自注意力捕获长程依赖与跨变量交互。

这类方法在高维非线性场景中通常优于传统方法，但也存在训练不稳定、超参数敏感、可解释性不足、对先验知识利用不足等问题。

### 1.2.4 现有研究不足与本文切入点

综合现有文献与本课题数据特点，仍有以下空缺：

1. 统计模型与深度模型融合方式往往“并联拼接”，缺少统一可解释方程骨架。
2. 对复杂缺失模式（连续缺失、空间相关缺失）的稳健性仍不足。
3. 在风速等物理驱动明显的变量中，地形与风向先验利用不充分。

据此，本文的核心切入点是“以可解释时空自回归方程为主干，以神经网络作为参数生成/分布约束模块”，形成两条递进研究路线：

1. 第三章：变系数统计模型 + GAN 对抗约束（VCAAN）。
2. 第四章：物理先验 + 时空对齐 Transformer 超网络（PG-STAT）。

## 1.3 本文研究内容与结构安排

### 1.3.1 本文研究内容

本文主要完成以下研究工作：

1. 构建并整理统一的时空插补问题定义与符号体系，覆盖两类任务（PM2.5、风速）。
2. 基于已发表工作完善 VCAAN 方法章节，系统梳理模型设计、对抗优化和实验结论。
3. 提出并实现 PG-STAT 模型：在双向时空自回归方程中嵌入物理对齐先验与 Transformer 超网络，实现一次前向非迭代插补。
4. 在多缺失模式、多缺失率设置下与统计和深度基线进行对比，分析模型优势与局限。

### 1.3.2 本文结构安排

全文共五章，结构如下：

1. 第一章介绍研究背景、研究现状、研究问题与本文工作。
2. 第二章给出理论基础，包括缺失数据机制、变系数自回归、GAN 与 Transformer。
3. 第三章介绍已发表的 VCAAN 方法及其在 PM2.5 数据上的实验结果。
4. 第四章介绍 PG-STAT 模型、实验设置与风速插补结果分析。
5. 第五章总结全文并提出后续研究方向。
